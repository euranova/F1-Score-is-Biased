utils.datasets module¶

Contains dataset-related functions.

utils.datasets.generate_toy_data(contamination_rate, radius, seed=None)¶

Generates a toy dataset. Normal samples are sampled from a standard gaussian distribution, anomalous samples are sampled from a noisy circle around the mean. There are 500 clean train samples, 1000 clean test samples and some anomalous test samples (see contamination_rate).

Parameters

  • contamination_rate – Float in [0, 1[; contamination rate of the test set

  • radius – Float; radius of the anomalous circle

  • seed – Int; seed for the dataset generation

Returns

(np.ndarray(Float), np.ndarray(Float), np.ndarray(np.uint8)); train set, test set, test labels

utils.datasets.load_dataset(dataset_name)¶

Loads the required dataset.

Parameters

dataset_name – String (in _DATASET_FILES); dataset required

Returns

(np.ndarray(Float), np.ndarray(np.uint8)); ordinal-encoded samples and corresponding labels (1 for anomalies, 0 for normal samples)

utils.datasets.normalize_data(fit_on, transform)¶

  • transform – tuple(np.ndarray(Float)); arrays of samples to normalise

Returns

tuple(np.ndarray(Float));; arrays of normalised samples

utils.protocols module¶

Contains the two algorithms presented in the paper as well as sub-functions used in these.

utils.protocols.algo1(samples, labels, test_size, model, compute_metrics=True)¶

Algorithm 1 as described in the paper.

Parameters

  • samples – np.ndarray(Float); samples to use to train and test the given model

  • labels – np.ndarray(np.uint8); corresponding labels (1 is anomalous)

  • test_size – Float; proportion of samples to use in the test set

  • model – sklearn model; model to train and evaluate

  • compute_metrics – Bool; whether to compute the metrics or not

Returns

if compute_metrics is True: (Float, Float, Float, Float); F1-score, AUC, AVPR and optimal-threshold F1-score if compute_metrics is False: (np.ndarray(Float), np.ndarray(Float), np.ndarray(Float)):

labels, scores and binary predictions of the test set

utils.protocols.algo2_end(x_train, x_test, y_test, model)¶
